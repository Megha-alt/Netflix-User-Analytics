# 🎬 Netflix User Analytics & Machine Learning

## 📌 Overview

This project analyzes Netflix user behavior and applies Machine Learning techniques to predict subscription renewals and monthly spending. The dataset contains user demographics, viewing habits, subscription details, and spending patterns.

## 📊 Dataset Features

* UserID
* Age
* Gender
* SubscriptionType
* WatchHoursPerWeek
* DevicesUsed
* FavoriteGenre
* AdClicks
* MonthlySpend
* SubscriptionRenewed

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Google Colab

## 🤖 Machine Learning Models

### Classification

* Decision Tree Classifier
* K-Nearest Neighbors (K=5)

**Target:** SubscriptionRenewed

### Regression

* Linear Regression

**Target:** MonthlySpend

## 🔄 Workflow

1. Data Loading
2. Data Cleaning
3. Missing Value Handling
4. Feature Encoding
5. Exploratory Data Analysis
6. Train-Test Split
7. Model Training
8. Model Evaluation

## 📈 Evaluation Metrics

* Accuracy Score
* Confusion Matrix
* MAE
* MSE
* R² Score

## 🚀 Key Outcomes

* Predicted subscription renewal behavior.
* Compared Decision Tree and KNN performance.
* Estimated monthly spending using Linear Regression.
* Generated insights from user engagement and subscription patterns.
